# 📝 Notes App (Next.js + LocalStorage)

A modern, responsive Notes CRUD application built using **Next.js (App Router)** and **Tailwind CSS**, with data persistence handled through **browser localStorage**.

This project was developed as part of an internship assignment to demonstrate frontend and application state management skills without relying on a backend database.

---

## 🚀 Live Demo

[(https://crud-application-three-ashen.vercel.app/](https://crud-application-three-ashen.vercel.app/)

---

## ✨ Features

* ➕ Create a new note
* 📄 View all notes
* 🔍 View a single note
* ✏️ Edit existing notes
* 🗑️ Delete notes
* 💾 Persistent data using localStorage
* 📱 Fully responsive UI

Each note includes:

* Title
* Content
* Creation timestamp

---

## 🛠️ Tech Stack

**Frontend**

* Next.js (App Router)
* React (Functional Components & Hooks)

**Styling**

* Tailwind CSS

**Data Storage**

* Browser localStorage (No backend required)

## 📂 Project Highlights

* Clean and modular component structure
* Efficient state management using React hooks
* Client-side data handling with localStorage

## ⚙️ Installation & Setup

```bash
git clone <your-repo-link>
cd notes-app
npm install
npm run dev
```

Open in browser:

```
http://localhost:3000
```

---

## ⚠️ Limitations

* Data is stored only in the browser (not shared across devices)
* No authentication system
* Not suitable for production-scale applications
